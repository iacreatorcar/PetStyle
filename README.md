# 🐾 PetStyle - E-commerce per Cani di Taglia Media

Benvenuti in **PetStyle**, un progetto di e-commerce moderno e responsive dedicato esclusivamente alla vendita di accessori e abbigliamento per cani di taglia media. Questo progetto nasce come parte del mio portfolio per dimostrare competenze nel design UI/UX e nello sviluppo Frontend.

---

## 🚀 Caratteristiche Principali

* **Vetrina Prodotti Dinamica:** Visualizzazione di accessori con sistema di filtraggio concettuale.
* **Carrello Interattivo:** Sistema di aggiunta prodotti gestito tramite JavaScript.
* **Blog Integrato:** Sezione dedicata a consigli utili per la cura dei cani (taglia media).
* **Design Responsive:** Ottimizzato per desktop, tablet e smartphone (Mobile First).
* **Form di Contatto:** Predisposto per l'invio di richieste via email.
* **Social Integration:** Link collegati ai profili social del brand.

---

## 🛠️ Tecnologie Utilizzate

* **HTML5:** Struttura semantica del sito.
* **CSS3:** Layout avanzato con Flexbox e Grid, variabili CSS per un design coerente.
* **JavaScript (ES6):** Gestione del carrello, manipolazione del DOM e interattività.
* **Google Fonts:** Per una tipografia moderna e leggibile.
* **FontAwesome:** Per le icone dei social e del carrello.
